# 🎵 Music Player - Spotify Style

Um reprodutor de música moderno com design inspirado no Spotify, desenvolvido com HTML, CSS e JavaScript puro.

## 📂 Estrutura do Projeto

```
music-player/
├── index.html           # Arquivo principal HTML
├── css/
│   └── style.css       # Estilos CSS (separado)
├── js/
│   └── script.js       # Lógica JavaScript (separado)
├── assets/
│   └── audio/          # Pasta para arquivos de áudio
└── README.md           # Este arquivo
```

## 🚀 Como Usar

1. **Abrir no navegador**
   - Abra o arquivo `index.html` no seu navegador preferido

2. **Adicionar suas músicas**
   - Coloque seus arquivos `.mp3` na pasta `assets/audio/`
   - Atualize a lista de `playlist` no arquivo `js/script.js`

3. **Personalizar a playlist**
   - Edite o array `playlist` em `js/script.js`
   - Formato: `{ title: 'Nome', artist: 'Artista', src: 'path/audio.mp3', emoji: '🎸' }`
